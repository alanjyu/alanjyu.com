Rock and physics!

Multi-scale modeling of the Rhine Graben

Using supercomputer models, I simulate long-term rifting (using geodynamic code ASPECT) and sedimentation (landscape evolution code Fastscape) that formed the Upper Rhine Graben, France–Germany. By modelling how rocks fracture and how heat and water flow through the porous crust (thermo-hydro-mechanical code GOLEM), I help identify target areas for geothermal energy development.
